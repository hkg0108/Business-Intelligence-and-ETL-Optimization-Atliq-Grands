# Business and Data Intelligence Enhancement at AtliQ Grands
🔍 Overview

AtliQ Grands, a luxury hotel chain, has been facing a decline in market share and revenue due to increased competition and non-data-driven decision-making.

This project focuses on leveraging Business Intelligence and Data Analytics to uncover actionable insights from historical booking data and support strategic decision-making.

🎯 Objective:

-> Analyze hotel performance across multiple cities and properties

-> Identify key drivers of revenue, occupancy, and customer behavior

-> Build interactive dashboards for stakeholders

-> Provide data-driven recommendations to improve business performance

🧰 Tech Stack:

-> Python (Pandas, NumPy) – Data processing & ETL

-> SQL – Data querying & transformation

-> Power BI – Dashboarding & visualization

-> Excel – Data cleaning & preprocessing

📂 Dataset:

Historical booking and hotel performance data
Covers multiple cities, properties, and booking platforms

Includes key business metrics like:

-> Revenue

-> Occupancy %

-> ADR (Average Daily Rate)

-> RevPAR (Revenue per Available Room)

⚙️ Project Workflow:

1. Data Preparation
   
 Cleaned and transformed raw datasets

 Handled missing values and inconsistencies

 Designed structured data models

2. Data Modeling
   
Created relationships between fact and dimension tables

Built metrics using calculated fields and aggregations

3. Analysis
   
Explored booking patterns, revenue trends, and customer behavior

Compared performance across cities and hotel types

4. Dashboard Development
   
Built interactive Power BI dashboards with filters and KPIs

Visualized:

-> Revenue trends

-> Occupancy patterns

-> Booking platform performance

-> City-wise insights

📈 Key Insights:

-> Revenue is concentrated in a few high-performing cities

-> Weekday vs weekend patterns significantly impact occupancy

-> Certain properties underperform despite high booking volume

-> Opportunities exist for pricing optimization and demand forecasting

💡 Business Recommendations:

-> Implement dynamic pricing strategies to maximize revenue

-> Focus on underperforming properties for operational improvement

-> Optimize booking platforms contributing lower revenue

-> Improve customer experience to increase repeat bookings.

🚀 Key Features:

-> End-to-end ETL pipeline + analytics workflow

-> Interactive dashboards for real-time insights

-> Business-focused storytelling with data

-> Scalable approach for future analytics use cases
